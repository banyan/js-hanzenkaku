# hanzenkaku.js

A Hankaku-Zenkaku translator in JavaScript.

## SYNOPSIS

### FUNCTIONAL INTERFACE

```js
import { toZenkaku } from 'hanzenkaku';

toZenkaku('ｺｶﾞｲﾀﾞﾝ'); // => コガイダン
```
